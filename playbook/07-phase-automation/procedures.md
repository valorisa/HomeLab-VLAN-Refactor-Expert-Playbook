# Phase 07 — Procédures

1. Créer un conteneur Debian (LXC) sur Proxmox.
2. Installer Docker + Compose.
3. Déployer Prometheus, node_exporter, blackbox_exporter, Grafana.
4. Installer unifi-poller pour stats Wi-Fi.
5. Installer Ansible, créer `inventory.ini`.
6. Écrire `backup-config.yml` pour récupérer les configs via SSH/API.
