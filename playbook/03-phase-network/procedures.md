# Phase 03 — Procédures réseau

1. Créer l'agrégat FortiGate (`config system interface`).
2. Créer les VLAN sub-interfaces (10,20,30,50,100,200,300,1000).
3. Activer le DHCP relay.
4. Sur Cisco : créer les VLANs, configurer le trunk, appliquer la liste d'accès.
5. Taguer les ports access selon `cisco-vlan-access-ports.ios`.
6. Vérifier LACP (`show etherchannel summary`).
