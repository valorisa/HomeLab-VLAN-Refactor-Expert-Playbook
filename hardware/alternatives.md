# Alternatives matérielles évaluées

## Pare-feu
- pfSense/OPNsense sur mini-PC x86 : + flexibilité, − nécessite tuning plus poussé.
- Ubiquiti UXG-Pro : + intégration UniFi, − LACP limité, pas de proxy mDNS.

## Switch cœur
- Aruba 2930F : + fanless sur certains modèles, − moins d'exemples communautaires.
- Dell N3048P : + uplinks 10G, − CLI différent.

## Switch 10G
- Ubiquiti USW-Enterprise-8-PoE : + ports cuivre, − prix élevé.
- Netgear XS708T : + ports 10G cuivre, − bruyant.

## Wi-Fi
- Aruba Instant On AP32 : + gestion cloud simple, − PPSK limité.
- TP-Link Omada EAP773 : + prix bas, − PPSK payant.

## Stockage / backup
- Synology DS923+ : + DSM, − coût neuf.
- Serveur Proxmox Backup dédié : + déduplication, − maintenance en plus.
